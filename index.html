Welcome! Below is a set of instructions for a simple calculator using JavaScript.

Instructions for Operating Calculator

  1. Righ click and open "inspector" or "inspect", then open the "console" tab. The calculations will be shown in the console using JavaScript.

  2. Without closing the console, refresh the browser and a prompt box will appear.

  3. Enter in one whole number in each prompt box, there will be two prompt boxes.

  4. In the console, the math operations for multiplaction, division, addition, and subtraction will be shown for the value you entered in step 3.
